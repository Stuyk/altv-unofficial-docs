# Time Flags

Controls when these lights are shown in-game.

| Value | Time In-Game |
| --- | --- |
| 1 | 1:00 AM |
| 2 | 2:00 AM |
| 3 - 11 | Repeats the same format as above for AM time. |
| 12 - 24 | Repeats the same format as above for PM time. |
| 25, 26, 27 | Lights are on 24/7. |
